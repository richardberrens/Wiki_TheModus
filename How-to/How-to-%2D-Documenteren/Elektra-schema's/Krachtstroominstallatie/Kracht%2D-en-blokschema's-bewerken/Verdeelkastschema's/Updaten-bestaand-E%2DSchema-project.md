# **Updaten bestaand E-Schema project**

Volg onderstaande stappen voor het updaten van een bestaand E-schema
project:
- Klik in 'Cadac Revit' > 'Productiviteit' > **[ ![themodus_prof_button_addin_productivity_documentation_16x16.png))](https://download.cadac.com/themodus/manual/themodus_prof_button_addin_productivity_documentation_16x16.png) | Documentatie ]** > **[ ![themodus_prof_button_addin_productivity_documentation_escheme_high_16x16.png))](https://download.cadac.com/themodus/manual/themodus_prof_button_addin_productivity_documentation_escheme_high_16x16.png) | Genereer E-schema ]**;
_'TheModus E-schema' wordt geopend._
- Open het opgeslagen E-Schema project door te klikken op de knop **[ ![themodus_generic_open_16x16.png))](https://download.cadac.com/themodus/manual/themodus_generic_open_16x16.png) | Openen ]**
    Browse naar het opgeslagen projectbestand (extentie .espx)
- Selecteer het bestand en klik **[ Openen ]**
_Er volgt een melding: 'E-schema project is ingelezen.'_
- Klik **[ OK ]**
- Update de bestaande project data met de nieuwe BIM data door te klikken op de knop **[ ![themodus_generic_refresh_16x16.png))](https://download.cadac.com/themodus/manual/themodus_generic_refresh_16x16.png) | Updaten schema ]**
_Er volgt een melding: ' "Project xxx" wordt ge-update met Revit model "Project xxx". Doorgaan?'_
Klik:
  - **[ Ja ]**
Er volgt een melding:
'Update voltooid'
of
'Update voltooid. De volgende kasten zijn vervangen of verwijderd in het model: '
    - Klik **[ OK ]** 
  - **[ Nee ]** om het updaten te stoppen.

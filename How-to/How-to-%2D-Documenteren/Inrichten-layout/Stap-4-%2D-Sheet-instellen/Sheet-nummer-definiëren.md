# **Sheet nummer definiëren**
Je kan een 'Sheet' automatisch voorzien van een nummer, gebaseerd op eigenschappen van de geselecteerde elementen en gebruikte viewtypen.

Je kan het sheetnummer instellen. 

**Werkwijze voor het instellen van het sheetnummer:**
- _Binnen  de sectie 'Blad instellen':_
- Selecteer achter 'Sheet nummer prefix' de projectparameter welke een waaarde bevat voor een voorvoegsel. Je mag hier ook een waarde typen
- Geef bij 'Sheet nummer start' het startnummer op. Je kan voorafgaand aan dit startnummer een symbool opgeven, bijv. "100". De bladnummering start nu bij nummer 100.

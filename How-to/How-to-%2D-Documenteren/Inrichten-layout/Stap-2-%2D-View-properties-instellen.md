# **View properties instellen**
In de 'View properties'Â worden specifieke weergave eigenschappen opgegeven zoals de naam, de fase, eventuele filters en nog overige eigenschappen van de view.

Je kan de volgende instellingen verwerken:
- Instellen van een gestandaardiseerde 'View' naam op basis van eigenschappen van de elementen;
- Instellen van een 'View' filter. Deze filter wordt dan toegevoegd aan uw view;
- Instellen van een 'View' fase;
- Instellen van de 'View' marges rekening houdend met het element;
- Instellen van de algemene 'View' kenmerken.

# **Eigenschappen van objecten aanpassen**
Met de TheModus Objecten Configurator kunt u eenvoudig eigenschappen aanpassen, waarbij direct het resultaat wordt getoond.

**Werkwijze voor het wijzigen van een object met de configurator:**
- Selecteer één object in een Revit project;
- Klik in 'Modify' > 'Cadac Revit bewerk' > **[ ![Object configureren.png))](https://download.cadac.com/themodus/manual/themodus_prof_button_addin_modify_configure_16x16.png) | Object configureren ]**;
- Wijzig de gewenste parameters;
- Klik **[ OK ]**
_Het object wordt gewijzigd met een nieuwe typenaam._

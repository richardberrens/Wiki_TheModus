# **Openen Object configurator**
De TheModus Objecten Configurator ondersteunt bij het maken van nieuwe typen van een familie. De configurator geeft direct visuele ondersteuning van de opgegeven resultaten. Afhankelijk van het object wordt de configurator ondersteund met een maatschets. 

De TheModus Objecten Configurator kan handmatig worden aangeroepen. Er zijn ook objecten waar TheModus de configurator automatisch aanroept vanuit het zoekscherm. Deze objecten zijn herkenbaar aan ![Configuratie.png))](https://download.cadac.com/themodus/manual/themodus_prof_button_configure_16x16.png) (Object configureren) in de afbeelding van het object.

**Werkwijze voor het openen van de configurator:**
- Selecteer één object in een Revit project;
- Klik in 'Modify' > 'Cadac Revit bewerk' > **[ ![Object configureren.png))](https://download.cadac.com/themodus/manual/themodus_prof_button_addin_modify_configure_16x16.png) | Object configureren ]**;
_Het configuratie scherm wordt geopend.
Het configuratiescherm is ingedeeld in 4 kolommen._
  1. De eerste kolom toont, indien beschikbaar, de maatschets. Deze geeft beknopt weer hoe een parameter geometrie aanstuurt;
  2. De tweede kolom toont het actieve type en alle eigenschappen ofwel parameters van een Revit familie. Indien het geselecteerde object bij TheModus is gemarkeerd als Configurabel Object dan toont de lijst vaak alleen de meest relevante parameters. Voor andere objecten worden alle parameters getoond. Aan de onderzijde wordt indien beschikbaar een toelichting van de parameter gegeven. Bij wijzigen van een parameter en verlaten van de cel, wordt het resultaat van de wijziging meteen getoond in de 4e kolom;
  3. De derde kolom bevat functionaliteiten voor connectoren te bewerken. Zie hiervoor de volgende paragraaf;
  4. De vierde kolom toont de viewer. In deze viewer wordt alleen het actieve element getoond. U kunt hier draaien en zoomen zoals u in een standaard view van Revit gewend bent. De viewer toont meteen de wijzigingen van het object.

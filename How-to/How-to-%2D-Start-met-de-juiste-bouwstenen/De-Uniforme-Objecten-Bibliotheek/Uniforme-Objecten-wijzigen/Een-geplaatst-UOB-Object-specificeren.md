# **Een geplaatst generiek object specificeren**

U kunt de informatie van een geplaatst generiek object in CAD uitbreiden met meer informatie. Zo kan de data gedurende de ontwerpfasen aangevuld worden wanneer dit noodzakelijk is. 

**Werkwijze:**
- Selecteer in Autodesk Revit het gewenste object;
- Klik 'Cadac Revit' > 'Uniforme Objecten' > **[ ![image1.png))](https://download.cadac.com/themodus/manual/themodus_nl_knop_uob_wijzigen_16x16.png)  Wijzigen ]**;
*De Uniforme Objecten Bibliotheek wordt geopend met daarin alle features, welke zijn verdeeld in "Product identificatie" alsmede "ETIM Kenmerken". Bij de Productvelden zijn alle velden beschikbaar die iets zeggen over het product, zoals fabrikant of artikelnummer. Als er sprake is van een Generiek Object, worden doorgaans deze velden niet gebruikt. De ETIM Kenmerken zijn verdeeld in diverse groepen. Klik op de groep om de gewenste features te tonen. Pas één of meer features aan naar wens. Alle features die relevant zijn voor het BIM-project moeten zijn aangevinkt. Voor alle niet aangevinkte features worden default waarden ingevuld, zodat het object netjes getekend kan worden.*
- Wijzig de waarden die aangepast moeten worden.
- Klik boven op **[ Overnemen ]**
*De nieuwe informatie wordt geladen in het model;*
*Het scherm met de typenaam van het product verschijnt.*
- Klik **[ OK ]** om het object bij te werken.
*of*
pas de typenaam aan als u een nieuw type wilt laden en klik **[ OK ]**

**N.B.**
Voor de betekenis van alle features en het invullen hiervan is extra ondersteuning beschikbaar. Klik **[ ![image1.png))](https://download.cadac.com/themodus/manual/uob_nl_knop_mcviewer_95x16.png) ]** -> **[ View only mode ]** om een maatschets aan te roepen van de ETIM klasse. Deze maatschets toont duidelijk waar iedere feature voor dient.

Met de knop **[ ![image1.png))](https://download.cadac.com/themodus/manual/uob_nl_knop_wijzigen_28x16.png) ]** kunnen waarden eenvoudig worden aangepast. Klik op **[ Overnemen ]** om terug te keren naar de feature lijst.

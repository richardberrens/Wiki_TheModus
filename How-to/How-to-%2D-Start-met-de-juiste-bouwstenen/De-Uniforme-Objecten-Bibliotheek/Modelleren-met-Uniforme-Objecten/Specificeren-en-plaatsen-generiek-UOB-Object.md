# **Een object specificeren en plaatsen in het BIM-model**
U kunt een object specificeren alvorens dit te plaatsen in het BIM-model. We noemen dit een Generiek Object.

Een Generiek Object bestaat uit data welke een redelijke representatie geeft aan het te plaatsen object. Deze representatie kan betrekking hebben op zowel vorm als functie. De mate van detail hangt volledig af van de wens van de modelleur. Dit is onder andere afhankelijk van de fase van het project, de fase van het object of de mate waarin informatie bekend is. Echter zijn er vele andere factoren die mede bepalen welke informatie op enig moment relevant is, en welke niet.

Het generieke object geeft een representatie van het product "in wording", en kan worden gebruikt voor analyse, voorbereidingen en calculatie.

**Werkwijze:**
- Klik 'Cadac Revit' > 'Uniforme Objecten' > **[ ![image1.png))](https://download.cadac.com/themodus/manual/themodus_nl_knop_uob_toevoegen_16x16.png)  Toevoegen ]**;
_Het zoekscherm wordt geopend._
- Gebruik de zoekfuncties om het gewenste object te zoeken;
- Klik bij de gewenste klasse op **[ ![image1.png))](https://download.cadac.com/themodus/manual/uob_nl_knop_wijzigen_28x16.png)  Specificeer ]**
_Een lijst met één of meerdere modelklassen opent;_
- Klik op de gewenste modelklasse;
_Een scherm opent met daarin alle features, welke zijn verdeeld in "Product identificatie" alsmede "ETIM Kenmerken".
Bij de Productvelden zijn alle velden beschikbaar die iets zeggen over het product, zoals fabrikant of artikelnummer. Als er sprake is van een Generiek Object, worden deze velden doorgaans niet gebruikt. 
De ETIM Kenmerken zijn verdeeld in diverse groepen. Klik op de groep om de gewenste features te tonen. Pas één of meer features aan naar wens. Alle features die relevant zijn voor het BIM-project moeten zijn aangevinkt. Voor alle niet aangevinkte features worden default waarden ingevuld, zodat het object netjes getekend kan worden._
- Selecteer 'Generiek object' bij 'Overnemen als';
- Markeer de velden die u wenst aan te passen;
- Pas de gewenste velden aan;
- Klik boven op **[ Overnemen ]**
_Het generieke product wordt geladen in het BIM-model;_
_Het scherm met de typenaam van het product verschijnt._
- Pas, indien gewenst, de typenaam aan en klik **[ OK ]**
*Het generieke object is nu geladen in het BIM-model. Dit object blijft aan de kruisdraad hangen, en kan daarmee meteen geplaatst worden.*

**Opmerking**
Voor de betekenis van alle features en het invullen hiervan is extra ondersteuning beschikbaar. Klik
op **[ UOB MC-viewer ]** en **[ Edit data mode ]** om een maatschets aan te roepen van de ETIM klasse. Deze maatschets toont duidelijk waar iedere feature voor dient.
Pas de gewenste waarden aan en klik **[ Overnemen ]** om terug te keren naar de feature lijst.

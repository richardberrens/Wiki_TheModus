# **Plaatsen van een generiek UOB Object**
U kunt een generiek UOB Object plaatsen in het BIM-model.

Een generiek UOB Object, ofwel Revit Family, is voorzien van zeer basale informatie, zodat het object in Revit getekend kan worden.

U kunt het object gebruiken om uw BIM-data voor dit object op te bouwen. In een later stadium kan deze informatie gebruikt worden om specifieke producten te zoeken.

**Werkwijze:**
- Klik 'Cadac Revit' > 'Uniforme Objecten' > **[ ![image1.png))](https://download.cadac.com/themodus/manual/themodus_nl_knop_uob_toevoegen_16x16.png)  Toevoegen ]**;
_Het zoekscherm wordt geopend._
- Gebruik de zoekfuncties om het gewenste object te zoeken;
- Activeer het tabblad **'Klassen'**, mits deze tab nog niet is geopend;
- Klik bij de gewenste klasse **[ ![image1.png))](https://download.cadac.com/themodus/manual/uob_nl_knop_download_28x16.png)  Downloaden]**;
_Een lijst met één of meerdere modelklassen opent;_
- Klik op de gewenste modelklasse;
_Het object wordt geladen en het scherm met de typenaam van het object verschijnt._
- Pas indien gewenst de typenaam aan en klik **[ OK ]**

_Het object is nu geladen in het BIM-model. Dit object blijft aan de kruisdraad hangen, en kan daarmee meteen geplaatst worden._

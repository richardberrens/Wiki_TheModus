# **Plaatsen van een specifiek UOB Object**
U kunt een specifiek UOB Object plaatsen in het BIM-model.

Een specifiek UOB Object, ofwel product, is voorzien van alle productinformatie, en daarmee een bestelbaar product.

U kunt het object gebruiken om uw bestelproces op te starten.

**Werkwijze:**
- Klik 'Cadac Revit' > 'Uniforme Objecten' > **[ ![image1.png))](https://download.cadac.com/themodus/manual/themodus_nl_knop_uob_toevoegen_16x16.png)  Toevoegen ]**;
_Het zoekscherm wordt geopend._
- Gebruik de zoekfuncties om het gewenste object te zoeken;
- Activeer het tabblad **'Producten'**, mits deze tab nog niet is geopend;
- Klik bij het gewenste product **[ ![image1.png))](https://download.cadac.com/themodus/manual/uob_nl_knop_download_28x16.png)  Downloaden]**;
_De download starten het scherm met de typenaam van het product verschijnt._
- Pas indien gewenst de typenaam aan en klik **[ OK ]**

_Het product is nu geladen in het BIM-model. Dit object blijft aan de kruisdraad hangen, en kan daarmee meteen geplaatst worden._

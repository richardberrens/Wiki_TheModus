# **Validatie rapport opvragen**
Alle resultaten van de validatie worden beschreven in een rapport. Dit rapport is online beschikbaar.
Dit rapport helpt niet alleen om de kwaliteit van het BIM-model te bewaken. Het helpt ook bij het bestellen van artikelen.

**Werkwijze:**
- Klik 'Cadac Revit' > 'Uniforme Objecten' > Pijltje onder **[ ![image1.png))](https://download.cadac.com/themodus/manual/themodus_nl_knop_uob_valideren_16x16.png)  Valideren ]** > **[ ![image1.png))](https://download.cadac.com/themodus/manual/themodus_nl_knop_uob_validatieverwerken_16x16.png)  Validatie verwerken]**;
_Het validatie resultatenscherm wordt geopend._
![image1.png))](https://download.cadac.com/themodus/manual/themodus_nl_dialoog_uob_validatieverwerken.png)
- Klik achter de aangevraagde validatie op: **[ ![image1.png))](https://download.cadac.com/themodus/manual/themodus_nl_knop_uob_zoeken_rapport_16x16.png)  ]**
_Het rapport wordt geopend. Het rapport toont:_
  - Koptekst met daarin de ticketgegevens;
  - Identificatie nummer van het gecontroleerde Uniform Object
  - Validatie ETIM structuur, waarin wordt weergegeven of de features van het object correct zijn, en met welke ETIM versie deze is vergeleken;
  - Validatie featuredata, waarin wordt weergegeven of het object voldoet aan de oorspronkelijke productgegevens. Deze validatie is alleen van toepassing voor specifieke objecten!
  - Validatie bestelbaarheid, waarin wordt weergegeven in welke mate het object nog bestelbaar is;
  - Tabel met de controle resultaten van dit object, met daarin de kolommen:
    - Featurecode, welke de code van de gecontroleerde feature weergeeft zoals deze bekend is bij ETIM.
    - Poortcode, welke weergeeft bij welke poort de gecontroleerde feature hoort.
    - Status, welke weergeeft of de feature met waarde overeenkomt met het artikel.
    - Waarde CAD, welke waarde in het CAD object van toepassing is.
    - Eenheid CAD, wat de bijbehorende eenheid van de feature is. 
    - Waarde online, welke waarde bekend is in de Uniforme Objecten Bibliotheek.
    - Omschrijving, de omschrijving van de waarde indien de waarde een grenswaarde betreft (ETIM EV code)
    - Eenheid online, de bijbehorende eenheid in de Uniforme Objecten Bibliotheek.
- Nadat u het rapport bestudeert heeft, kunt u het rapport afsluiten door te klikken op **[ X ]** (kruisje).
- Sluit het validatie formulier door te klikken op **[ Afsluiten ]**

**Opmerking**

Indien gewenst kunt u de tekst selecteren en kopiëren naar het klembord voor andere toepassingen

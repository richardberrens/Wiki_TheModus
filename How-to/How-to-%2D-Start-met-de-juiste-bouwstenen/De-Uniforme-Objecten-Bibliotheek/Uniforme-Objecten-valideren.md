# **Valideren Uniforme Objecten**
U kunt Uniforme Objecten valideren op onderstaande onderdelen:
- Controleer of een geplaatst product nog leverbaar is;
- Controleer of de informatie van een geplaatst object nog conform de opgave van de fabrikant is;
- Controleer of de structuur van een geplaatst object nog conform de specificaties van ETIM zijn.

Resultaten van een rapportage worden in een gedetailleerd rapport teruggegeven. Resultaten kunnen ook in het Autodesk Revit model worden geladen. Hiermee kunnen ze meteen visueel worden geaccentueerd.
